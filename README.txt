This is a piar project, to create a simple task manager for an assignment in module CP3101B Web Development in NUS.
The list of available functions to a user is as follows:
- Register
- Log in
- Add Tasks
- Edit Tasks
- Delete Tasks
- Update Tasks

-------------------------------------------

Installation instructions:

1. An Apache server will be required prior to installation of files.
2. Postgresql capability should also be enabled in PHP.
3. Setup database as per schema.sql
4. Edit config.inc with appropriate Postgresql database name, username and password
